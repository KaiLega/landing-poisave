# PoiSave — Landing Page

Landing page for the PoiSave mobile app, built with React + Vite + Tailwind. It showcases the product hero, feature highlights, screenshots, FAQ, and download call-to-action.

## Tech stack
- React + TypeScript
- Vite for dev/build
- Tailwind CSS for styling
- Lucide icons

## Structure
- `src/App.tsx` — main page sections (hero, features, how it works, screens, FAQ, download).
- `src/components/` — reusable bits (`Header`, `Footer`, `Feature`, `FAQ`).
- `src/styles.css` — Tailwind setup plus custom utilities/variables.
- `public/` — static assets (screens, logos, badges).
